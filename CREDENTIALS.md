# Учетные данные для входа в систему

## Роли пользователей

- **guest** (Гость) - роль по умолчанию, не требует авторизации
- **owner** (Владелец) - требуется авторизация с паролем
- **admin** (Администратор) - требуется авторизация с паролем

## Пароли для входа

| Роль | Пароль |
|------|--------|
| Владелец (owner) | `owner2026-rental` |
| Администратор (admin) | `admin2026-rental` |
